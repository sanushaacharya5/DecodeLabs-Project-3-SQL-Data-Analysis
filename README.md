# DecodeLabs Project 3 - SQL Data Analysis

## Objective
Analyze an e-commerce dataset using SQL to extract business insights.

## Tools Used
- MySQL Workbench
- Excel

## Queries Performed

### 1. Total Orders
Count total orders in the dataset.

### 2. Product Sales Analysis
Identify the most frequently ordered products.

### 3. Orders by Payment Method
Analyze customer payment preferences.

### 4. Orders by Status
View the distribution of order statuses.

### 5. Average Product Price
Compare average prices across products.

### 6. Total Quantity Sold by Product
Identify products with the highest sales volume.

### 7. Referral Source Analysis
Determine which marketing channels generate the most orders.

### 8. Top 10 Most Expensive Orders
Display the highest-priced orders.

## Key SQL Concepts Used
- SELECT
- GROUP BY
- ORDER BY
- COUNT()
- SUM()
- AVG()
- LIMIT
